# LearnFlow Design Tokens

## Colors — Dark Mode (Primary)
- Background: #191919
- Surface/Card: #222222
- Border: #2e2e2e (0.5px)
- Accent: #95D1BD
- Accent Bright (pressed/active): #A5E1CB
- Text Primary: #EFEFEF
- Text Secondary: #8a8a8a
- Star/Rating: #F59E0B
- Destructive: #EF4444
- Offline Banner: #92400E on #451A03

## Colors — Light Mode
- Background: #F7F7F5
- Surface/Card: #FFFFFF
- Border: #E5E5E5
- Accent: oklch(52% 0.09 168.94)
- Text Primary: #1a1a1a
- Text Secondary: #6b7280

## Spacing
- Screen horizontal padding: 16px
- Card gap: 12px
- Section gap: 24px

## Border Radius
- Cards: 10–12px
- Buttons: 8px
- Pills/Tags: 6px

## Typography
- Headings: bold, 22–28px
- Body: 14–15px
- Eyebrow labels: 10–11px, uppercase, letter-spacing 0.08em
- Font: Inter or closest system geometric sans

## Style Rules
- Flat cards only — no glassmorphism, no heavy shadows
- 0.5px borders on all cards and inputs
- Notion-level whitespace, Udemy-level density on course screens
- Hierarchy via size and weight, not color overload
